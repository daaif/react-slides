JSX Jvascript XML

JSX est une extension ECMAScript utilisée par React. Elle permet la coexistance de texte de type XML / HTML avec le code JavaScript. La syntaxe est destinée à être utilisée par des préprocesseurs (Ex. Babel) pour transformer du texte de type HTML trouvé dans des fichiers JavaScript en objets JavaScript standard qu'un moteur JavaScript analysera.
